1. jsx란? 
- 자바스크립트에 XML을 추가한 문법
- 하나의 파일에 자바스크립트와 HTML을 동시에 작성
- 가독성이 높고 작성하기 쉬움
- 반드시 부모 요소 하나가 감싸는 형태여야 함 = 단일 결과물
     + 이유는? 컴포넌트 내부는 하나의 DOM트리 구조를 이루어야 하기 때문
- {} 를 통해 유효한 모든 자바스크립트 표현식 사용 가능

2. 컴포넌트 생성 방법
(1) 함수형 컴포넌트 기본 틀에 HTML코드 작성
(2) export와 import를 통해 재사용
(3) 재사용 시에는 <컴포넌트이름 />의 형태로 작성

3. props란?
- 상위 컴포넌트에서 하위 컴포넌트로 데이터를 전달해주는 "객체"
- props를 사용하면 상위 컴포넌트에서 데이터를 제어해 하위 컴포넌트에 전달 할 때 코드를 간결히 할 수 있음
